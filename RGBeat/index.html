RGBeat

RGBeat is an unorthodox music maker that coverts color to sound. Comprised of cellular rows, each circle represents a possible note to be played. Black circles are dead notes, while colored circles represent playable notes.

To activate notes, click a circle and drag to select your desired color. This color value is then used to create a unique synthesizer.

R = Octave of the Note Played

G = Sample Rate of the Synthesizer

B = Duration of the Note Played

Once your creation is complete, "RGBeat it". Feel free to change colors and update synthesizers as RGBeat plays.

This project expands upon the waveform audio synthesizer created by github user keithwhor.
