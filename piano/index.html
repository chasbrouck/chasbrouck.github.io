Piano

Piano is web audio experiment that allows you to create and array of instrumental sounds. The base of the product relies on a waveform audio synthesizer written in JavaScript by keithwhor (https://github.com/keithwhor/audiosynth). This JavaScript library allows the user to easily produce a waveform note of any musical key.

My Role

In this project I expanded upon keithwhor's open source library to create a responsive and customizable "web instrument". To start, I created a simplistic user interface which resembles the board of a standard piano. The user can press buttons (on mobile) or use the keyboard (on desktop) to play notes of a specific octave. Additionally, I wrote numerous algorithms which allow the user to heavily customize the sound of the "web instrument". This includes the attack, dampen rate, modulation and sample rate of the instrument. Try the piano out below!
