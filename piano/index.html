Piano

Piano is web audio experiment that allows you to create custom instruments. This project expands upon the waveform audio synthesizer created by github user keithwhor.

My Role

I created a simplistic user interface which resembles the board of a standard piano. The user can press desktop keys or utilize a touch screen to play notes. Additionally, I wrote numerous algorithms which allow the user to heavily customize the sound of the "web instrument". This includes the attack, dampen rate, modulation and sample rate of the instrument.
