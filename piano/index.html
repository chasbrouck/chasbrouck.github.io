Piano is web audio experiment that allows you to create custom instruments. This project expands upon the waveform audio synthesizer created by github user keithwhor.

Adjust sliders and settings to alter the attack, dampen rate, modulation and sample rate of your custom intrument. Then use the keyboard to play your custom instrument like a piano.

Live Site: http://chasbrouck.com/pianoplay/
